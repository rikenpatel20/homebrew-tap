# homebrew-tap

Homebrew formulae for [@rikenpatel20](https://github.com/rikenpatel20)'s projects.

## claude-statusbar

A macOS menu-bar dashboard light for [Claude Code](https://github.com/rikenpatel20/claude-statusbar).

```bash
brew install --cask swiftbar                      # menu-bar host (free)
brew install rikenpatel20/tap/claude-statusbar    # the tracker
claude-statusbar --write-settings                 # finish setup
```

Then restart Claude Code. See the
[project README](https://github.com/rikenpatel20/claude-statusbar) for details.
